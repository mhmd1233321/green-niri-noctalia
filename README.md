## 🛠 Prerequisites
To get this exact look, ensure you have the following packages installed:

- **WM:** [Niri](https://github.com/YaLTeR/niri)

- **Terminal:** Alacritty / Kitty

- **System Monitor:** Btop

- **Fonts:** [Nerd Fonts](https://www.nerdfonts.com/) (Required for icons)

- **Fetch:** Fastfetch

- **Audio Visualizer:** Cava

- **Matrix:** Cmatrix
